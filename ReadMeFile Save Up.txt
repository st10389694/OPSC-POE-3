# Save Up — Budget Tracker App

**Save Up** is a personal finance Android app built in Kotlin using Firebase as backend. It helps users manage expenses, track spending by category, and stay within monthly goals.

---

## 📱 Features

✅ User authentication (Sign Up / Log In)  
✅ Create categories for expense tracking  
✅ Add expenses with date, time, description, amount, category & photo  
✅ Set monthly min/max budget goals  
✅ View expenses for a selected time period  
✅ View Pie Chart of category spending  
✅ Export expense data to CSV  
✅ Dark/Light mode toggle  
✅ Badges for logging expenses and meeting goals (Gamification)

---

## 🔐 Firebase Services Used

- Firebase Authentication  
- Firebase Realtime Database  
- Firebase Storage (for storing receipt photos)

---

## 📊 Libraries Used

- [MPAndroidChart](https://github.com/PhilJay/MPAndroidChart) — for Pie Chart  
- Android Jetpack Navigation & View Binding  
- Firebase KTX SDKs

---

## 🚀 How to Run

1. Clone this repository in Android Studio.
2. Connect your Firebase project and download `google-services.json` into `app/`.
3. Sync Gradle and build the project.
4. Run on emulator or physical device (Android 7+).

---

## 🎖 Own Features

1. **Gamification**: Badges are awarded for logging 10+ expenses or staying within goals.
2. **CSV Export**: Pie chart data can be exported as CSV for offline usage.
